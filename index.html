Semantics (more information)

Section is use to make "element group" (topically related);
In Sections headers are not same as in head (h1 in head would not be same size as in section);

Header - could contain logo or navigation;
Header could be under section (section header);

Hgroup - group of headings;
Used if headings directly follow each other;
Rare use;

Nav - navigation section;
Suitable for central navigation class;
Used for links to other documents, links to anchors;

Article - independent are on the page;
Nesting is possible: each sub-article should be independent;

Aside - side element;
Block element - to make side - use CSS;

Footer - Meta information place;

Very simple example:

<DOCTYPE! html>
<head>
<title> This is title</title>
<!-- Here usually all links and script sources are put -->
</head>

<body>
<header>
<nav> </nav>
<!-- This can contain logo and navigation -->
</header>

<section>
<!-- Sections are used to order topically related groups -->
</section>

<aside>
<!-- Aside is used to tag element which would be aside (It is block element without CSS) -->
</aside>

<footer>
<!-- Footer is meta information place -->
</footer>
</body>
</html>
